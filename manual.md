# 英単語 アプリ

## 技術スタック

- Vite
- React
- CSS
- TypeScript

## データ

- source.md にデータが記載されています。

## 画面一覧

### ヘッダ

- アプリ名を表示します。

### 制作ルール

- TypeScript を使用します。
- 英単語のアプリを作ります。
- 最初は英語を表記させ、下部に「意味」という名前のボタンを押すことにより、日本語の意味を表示させます。
- 1画面に1つの問題を表示させます。
- 表示方法は英語のデータからランダムで表示させます。
- 表示方法は source.md に記載されている parts で品詞毎に区別されています。また、 grade で学ぶ学年を区別しています。これらを 種類ごとに分けています。parts 別、grade 別、それと、すべてを混ぜた完全なランダムをプルダウンボタンによって選ぶことができ、部類別、全ランダム単語表記にさせます。
- source.md には id も記載されていますので、id の表記も、お願いします。(例) id / 全id のように。
- 英単語に使う英語と日本語の意味のデータは、 source.md に記載されています。 e_word が英語、 mean が日本語の意味になります。
- アプリのデザインは、文字やボタンは大きく、見やすいデザインにしてください。
